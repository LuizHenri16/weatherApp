# WeatherApp 🌤️

Um aplicativo de previsão do tempo moderno e animado, desenvolvido com React Native e Expo.

## 🚀 Tecnologios Utilizadas

- **React Native** (0.83.2)
- **Expo** (~55.0) - Framework para desenvolvimento rápido e fácil
- **Expo Router** - Roteamento baseado em arquivos
- **TypeScript** - Tipagem estática para maior segurança e produtividade
- **React Native SVG** - Renderização de ícones e gráficos em SVG
- **React Native Linear Gradient** - Gradientes visuais

## 📦 Instalação e Execução

Siga os passos abaixo para rodar o projeto localmente:

1. Clone o repositório e acesse a pasta do projeto:
```bash
cd weatherApp
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento do Expo:
```bash
npm start
```

## 📱 Scripts Disponíveis

- `npm start`: Inicia o Metro Bundler do Expo.
- `npm run android`: Roda o aplicativo em um emulador ou dispositivo Android conectado.
- `npm run ios`: Roda o aplicativo em um simulador iOS (necessário macOS).
- `npm run web`: Roda o aplicativo no navegador.

## 📁 Estrutura do Projeto

A estrutura principal do projeto está localizada na pasta `src/`:
- `src/app/`: Arquivos de rotas do Expo Router (ex: `index.tsx`).

## 🛠️ Requisitos

- Node.js
- Aplicativo Expo Go no celular (para testar em aparelho físico) ou emulador Android/iOS configurado.

## 📱 Testando no Dispositivo Físico (Android)

Se você quiser apenas testar o aplicativo no seu dispositivo Android sem precisar rodar o servidor de desenvolvimento:
- Vá até a área de **Releases** deste repositório.
- Baixe o arquivo `.apk` mais recente.
- Instale-o diretamente no seu celular Android.
